I was working on rendering react sidebar because we structured our project nav bar on the side.

The crutial step to make it work is to place home-page nested routes (outlet) on the wherever layout you want the nested routes to be rendered (ie, in the project, place outlet in the Col. tag). Next is to create a nav data array so in the home-page, I can map through the array and render the route path by accessing it from the array.

Other concept I've learned from working on this feature is to have a 'page not found' route and 'protected route'.

For the 'protected route', I used ternary operation to allow access to the routes that are protected. Basically use Outlet to allow protected routes to show if true, and navigate to log in page if false.

The takeaway is to use all i've learned such as ternary operator, map array, nested routes and outlet'.
